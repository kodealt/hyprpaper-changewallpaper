# hyprpaper-changewallpaper
custom wallpaper changer wrapper-sort for myself, feel free to use or modify.

---
Download the file or copy and paste the code into a file (no extention)\n
copy the file to `/usr/bin`\n
use `sudo chmod +x /usr/bin/cwp` to make it executable\n
run `cwp help` for all commands\n
only config option currently supported is `path` which sets perma-path, a counterpart to the temp `set-path`\n

much love.
